Behind the Game

What is this?

A browser-based Countdown word game built as a coding practice project. The game uses a weighted letter picker backed by real English word frequency data, analysed with Python.

The Word List

The game validates words against the ENABLE dictionary — a public domain list of standard English words. The full list was filtered twice using Python to keep only words relevant to the game.

172,823 Original words
105,241 After length filter (2–9 letters)
96,963 After repeat filter (max 2 of each letter)

The second filter removes words where any letter appears more than twice — matching the game rule that no letter can be picked more than twice. This removed a further 8,278 words.

Letter Frequency Analysis

Each letter's probability was calculated by counting its occurrences across all 105,241 words. The picker uses these weights so common letters appear more often.

How Weighted Picking Works

Instead of picking randomly from a flat list, the game assigns each letter a weight proportional to its frequency. A random point is chosen along the total weight, and we walk through the letters until we hit it — meaning E (11.8%) is picked far more often than Q (0.18%).

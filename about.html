Behind the Game

What is this?

A browser-based Countdown word game built as a coding practice project. The game uses a weighted letter picker backed by real English word frequency data, analysed with Python.

The Word List

The game validates words against the ENABLE dictionary — a public domain list of standard English words. The full list was filtered three times using Python to keep only words that can actually be formed under the game rules.

172,823 Original words
105,241 After length filter (2–9 letters)
96,963 After repeat filter (max 2 of each letter)
94,554 After game limits filter (max 5 vowels, max 6 consonants)

Each filter mirrors a game rule — so every word in the final list can actually appear as a valid answer during play. The three filters together removed 78,269 words from the original dictionary.

Letter Weights — Raw vs Coverage

The first approach counted how many times each letter appeared across all words (raw frequency). The second counts how many different words each letter appears in (word coverage). Coverage produces a more useful distribution for gameplay.

Word Length Distribution

How many words exist at each length in the filtered dictionary. The peak at 8 letters means there are plenty of long word possibilities — but 9-letter words are rarer, making them worth double points.

How Weighted Picking Works

Instead of picking randomly from a flat list, the game assigns each letter a weight proportional to its frequency. A random point is chosen along the total weight, and we walk through the letters until we hit it — meaning E (11.8%) is picked far more often than Q (0.18%).
